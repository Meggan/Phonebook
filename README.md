# Phonebook
# Simple console phone book written in C

1. Add - Adds a new person to the phone book with first name, last name, and phone number
2. Retrieve - Finds a user from the phone book based on either 3 criterias.
3. Delete -  Deletes user based on either 3 criterias.
4. Load - Load phone book from specified txt file.
5. Save - Saves phone book.
